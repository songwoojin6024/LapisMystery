# Lapis Mystery

**Lapis Mystery**는 Windows 콘솔 환경에서 실행되는 C++ 추리 어드벤처 게임입니다.  
플레이어는 탐정이 되어 캠퍼스를 돌아다니며 도난당한 청금석 사건의 단서를 모으고, 용의자들과 대화하거나 미니게임을 클리어하며 최종 범인을 추리해야 합니다.

## 게임 소개

계명대학교의 상징과도 같은 청금석이 사라졌다.  
사건 현장에는 여러 수상한 흔적들이 남아 있고, 용의자들은 각자 의심스러운 정황을 가지고 있다.

플레이어는 도서관, 공과대학, 본관, 카페(상점) 등을 탐색하며 단서를 모아야 한다.  
충분한 단서를 확보한 뒤 최종 추리를 통해 진범을 지목하면 굿엔딩으로 이어지고, 잘못된 선택을 하거나 체력이 모두 소진되면 배드엔딩을 맞이하게 된다.

## 주요 기능

- 콘솔 기반 오픈월드 탐색
- WASD / 방향키 이동
- NPC 대화 및 미니게임 진행
- 단서 수집 시스템
- 진행도 표시 시스템
- 체력 및 돈 시스템
- 카페 상점 시스템
- 인벤토리 시스템
- 미니게임 패스 아이템
- 최종 추리 및 범인 지목
- 굿엔딩 / 배드엔딩 분기

## 조작법

| 키 | 기능 |
|---|---|
| W / ↑ | 위로 이동 |
| S / ↓ | 아래로 이동 |
| A / ← | 왼쪽 이동 |
| D / → | 오른쪽 이동 |
| I | 인벤토리 열기 |
| Enter | 선택 / 진행 |
| ESC | 나가기 / 취소 |

## 게임 지역

| 지역 | 설명 |
|---|---|
| 월드맵 | 캠퍼스 전체 이동 |
| 도서관 | 검색 기록, 도서관 NPC 단서 확인 |
| 공과대학 | 용의자 NPC와 주요 단서 조사 |
| 본관 | 경비실, CCTV, 최종 추리 진행 |
| 카페 | 아이템 구매 및 회복 |

## 미니게임

NPC와 상호작용하면 다양한 미니게임을 진행할 수 있습니다.

- 카드 맞추기
- 가위바위보
- 사다리타기
- 베스킨라빈스 31 방식 숫자 게임
- 틱택토
- 행맨
- 업다운 번호 맞추기

미니게임에서 승리하면 단서와 보상을 얻을 수 있습니다.  
카페에서 구매한 **뇌물용 초콜릿**을 사용하면 원하는 미니게임을 한 번 패스할 수 있습니다.

## 상점 및 인벤토리

카페에서는 돈을 사용해 아이템을 구매할 수 있습니다.

| 아이템 | 효과 |
|---|---|
| 익명의 제보 서류 | 송우진의 신체 정보 단서 획득 |
| 증거 분석용 돋보기 | 교수의 방 분석 단서 획득 |
| 뇌물용 초콜릿 | 미니게임 1회 패스 |
| 레쓰비 캔커피 | 하트 1 회복 |
| 매점 핫도그 | 하트 3 회복 |
| 박카스D | 하트 전체 회복 |

## 엔딩

게임에는 두 가지 엔딩이 존재합니다.

- **Good Ending**  
  충분한 단서를 모으고 최종 추리에서 진범을 정확히 지목하면 도달합니다.

- **Bad Ending**  
  체력이 모두 소진되거나 최종 추리에서 잘못된 범인을 지목하면 도달합니다.

## 프로젝트 구조

```txt
ProjectOOP/
├─ main.cpp                 # 게임 메인 루프
├─ GameManager.h            # GameState, Player, MapData, Item 정의
├─ Game.cpp                 # 공통 입력, 진행도 함수
├─ MapStates.cpp            # 월드맵/도서관/공대/본관/카페 상태 처리
├─ NPCInteraction.cpp       # 미니게임 NPC 상호작용
├─ NPC.h                    # NPC 클래스 및 미니게임 NPC 정의
├─ DialogueNPC.h            # 대화 전용 NPC UI
├─ DialogueNPCData.cpp/.h   # 대화 NPC 데이터
├─ InvestigationObjects.h   # 조사 오브젝트 단서 이벤트
├─ FinalDeduction.h         # 최종 추리 및 범인 지목
├─ GoodEnding.h             # 굿엔딩 연출
├─ BadEnding.h              # 배드엔딩 연출
├─ opening.h                # 오프닝 연출
├─ Maps.cpp/.h              # 맵 데이터
├─ Common.cpp/.h            # 콘솔 초기화 및 화면 출력
├─ CardGame.h               # 카드 맞추기 미니게임
├─ face1.h / face2.h        # 스토리 얼굴 아스키 아트
```
